In accordance with applicable FINRA rules, user will need to obtain a letter written and signed by a compliance officer on company letterhead explicitly granting permission for the applicant to carry the account and explicitly stating whether or not the firm requires access to duplicate account statements and/or trade confirmations. If the firm would like online access your duplicate account statements and/or trade confirmations, include in the letter the applicant's full name and the following information for the interested party: First and last name, title, email address, telephone number. A PDF version of the letter must be uploaded with the application. For any affiliated person, you will also have to upload a document separately via the Accounts API document upload endpoint with the following information about the company: name, address, compliance email address and the ticker symbol if it is a public company.
